# Cisco-Packet-Tracer-project-04-Dual-Router-Branch-Office-Network-2Routers-2Switches-Server-PC-Laptop
I’m happy to share my latest networking project – a complete branch office simulation built in Cisco Packet Tracer. The topology includes two Cisco 2911 routers (Router2, Router3), two 2960 switches (Switch0, Switch1), a central server (Server0), a desktop PC (PC0), and a laptop (Laptop0).

![image alt](https://github.com/Sameera54321/Cisco-Packet-Tracer-project-04-Dual-Router-Branch-Office-Network-2Routers-2Switches-Server-PC-Laptop/blob/main/5.png?raw=true)

## 📌 Summary

### Branch Office Network Lab is a Cisco Packet Tracer simulation of a typical small enterprise or branch office. It includes:

    2 x Cisco 2911 routers – providing connectivity between two LAN segments and/or WAN uplink

    2 x Cisco 2960‑24TT switches – access layer switches connecting end devices

    1 x Server – offers network services (DHCP, DNS, web, file)

    2 x end devices (PC, Laptop) – clients accessing server and internet

### The lab focuses on:

    Configuring static routes or a dynamic routing protocol (RIP/OSPF/EIGRP)

    VLAN creation and inter‑VLAN routing

    Switch port security and Spanning‑Tree Protocol

    Server service configuration (DHCP to automate IP assignment)

    Basic network troubleshooting (ping, traceroute, show commands)

This simulation serves as a hands‑on reference for CCNA‑level networking skills.

## ✨ Features

    ✅ 2 Cisco 2911 routers – fully configurable via CLI

    ✅ 2 Cisco 2960 switches – support for VLANs, trunking, port security

    ✅ Server0 – pre‑configured for DHCP, DNS, HTTP, FTP (or as needed)

    ✅ PC0 and Laptop0 – obtain IPs via DHCP or static assignment

    ✅ Inter‑VLAN routing – router‑on‑a‑stick or routed ports

    ✅ Dynamic routing (optional) – OSPF or EIGRP between Router2 and Router3

    ✅ Basic security – ACLs, SSH, enable passwords

    ✅ Full Packet Tracer file (.pkt) – ready to open and practice

    ✅ Documentation – IP addressing table, VLAN mapping, configuration excerpts

## 🛠️ Built With

    Cisco Packet Tracer – version 8.x (or later)

    CLI – all device configurations

    (Optional) Text editor – for documenting configs
