# Cisco-Packet-Tracer-project-04-Dual-Router-Branch-Office-Network-2Routers-2Switches-Server-PC-Laptop
I’m happy to share my latest networking project – a complete branch office simulation built in Cisco Packet Tracer. The topology includes two Cisco 2911 routers (Router2, Router3), two 2960 switches (Switch0, Switch1), a central server (Server0), a desktop PC (PC0), and a laptop (Laptop0).
